# facial-emotion-recognition
MATLAB behavioral experiment studying how facial orientation and occlusion affect emotion recognition speed and accuracy. Includes stimulus presentation and ANOVA analysis. 

**Course:** Research Methods in Psychology, UC Santa Cruz  
**Tools:** MATLAB

## Overview
A behavioral experiment examining how facial orientation (upright vs. inverted) 
and occlusion (masked vs. full) affect the speed and accuracy of emotion recognition.

## What the code does
- Presents randomized happy/sad face stimuli across 4 conditions
- Captures keypress responses and reaction times
- Outputs per-participant .mat data files
- Runs a two-way repeated-measures ANOVA across all participants
- Produces bar charts with error bars for accuracy and reaction time

## Key finding
Occlusion had a statistically significant effect on recognition accuracy.
